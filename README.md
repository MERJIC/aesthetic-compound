# 审美复利

一款围绕「审美 → 作品 → 粉丝 → 金钱 → 审美」构建的纯数值增量游戏。

新游戏从 ¥5,000 和两条完整生产线开始。本轮累计收入达到 ¥100 万后，可以重生并获得永久强化每层产出的「审美遗产」。

选题和灵感是会被创作持续消耗的库存，使用热点审美和深度审美按固定价格补充。「长期审美研究」采用近乎不递减的升级曲线，为两条生产线提供长期源头增长。

V7 使用弹性周转：库存越多，同时生产的作品越多，不再存在固定创作上限。选题和灵感默认自动收集，各节点里程碑分别强化收集、周转、传播与变现。

V8 将审美遗产改为平方根源头加成，遗产获取采用对数收益并随现有遗产提高门槛。「长期审美研究」改为线性产出增长，控制后期数值膨胀。

V9 让流量分配产生核心粉导流，让风格分配产生广告溢价。作品数量压缩为原来的五分之一，同时补偿曝光、口碑与作品反馈，维持原有总效益。闲置金钱会产生利息，并可通过「现金管理」升级。

V10 加入五步内嵌式新手指引：亲手完成选题、灵感、自动收集、源头扩建和审美分配。新档的自动收集默认关闭；指引结束后，同一区域会持续显示当前瓶颈建议。老存档不会被强制重走，可随时点击「玩法」重新打开。

V11 将底部单一节点改为顶部任务板，同时展示三个自动结算的短目标。24 项任务覆盖作品、传播、粉丝、扩建、升级、利息与本轮收入，单次奖励降低为 5%—10% 的局部强化，并通过更多阶段持续叠加。

动态瓶颈提示会在完成 8 项基础任务后自动隐藏，避免中后期继续占据页面；「玩法」按钮仍可随时重新打开五步指引。
